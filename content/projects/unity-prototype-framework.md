## 项目简介

这是一个面向玩法验证的 Unity 项目模板，用于快速测试角色控制、输入处理、场景流程和可复用玩法系统。

## 技术栈

- Unity
- C#
- ScriptableObject 架构
- 面向 GitHub Pages 的 WebGL 导出规划

## 计划功能

- 模块化角色控制器
- 场景启动流程
- 用于调参的调试 UI
- 可嵌入网页的 WebGL 构建页面
